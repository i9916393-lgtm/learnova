Skill Badges — How They Work

After both students complete a swap session, the platform generates a verified Skill Badge for each participant. The badge includes the skill name, partner's name, date, and a unique ID. These badges appear on your public profile and can be downloaded for LinkedIn or resumes.

Live Chat — jQuery Messaging

Built with jQuery's DOM manipulation and event system, the chat interface lets matched students message each other in real time. Share code snippets, links, and session notes. All messages are stored locally for the session — no server required for the static demo.

jQuery Powered Real-time UI Code Sharing
Swap Scheduler — Book Sessions

The scheduler lets both students pick mutually available time slots. Bootstrap's date/time components power the UI. Once confirmed, the session appears on both profiles with a countdown timer. Students can reschedule or cancel with one click.

Mutual Slots Countdown Timer One-click Reschedule
Portfolio Builder — Showcase Your Work

Every collaborative project you complete gets added to your LearnNova Portfolio. The builder auto-fills project title, collaborators, skills used, and date. Your portfolio page is public and shareable — a living proof of your growth that no certificate can replace.
